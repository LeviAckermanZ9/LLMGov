# Architecture Decision Records
This directory contains Architecture Decision Records (ADRs) for LLMGov.

Each ADR documents a significant technical decision, its context, the options
considered, and the rationale for the chosen approach.

## Naming Convention
- `ADR-NNN-short-title.md` (e.g., `ADR-001-litellm-provider-abstraction.md`)

## Template
See the [ADR template](https://adr.github.io/) for the standard format.
